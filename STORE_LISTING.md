**자세한 설명** (한국어):

```
환율 변환기는 30종 이상의 통화를 실시간으로 변환하고, 한 번 조회한 환율은
30분 동안 기기에 캐시되어 오프라인에서도 바로 사용할 수 있는 가벼운 금융
유틸리티입니다.

주요 기능
• 30종 이상의 통화 — 유럽중앙은행(ECB) 공식 데이터 (frankfurter.app)
• 30분 환율 캐시 — 네트워크가 끊겨도 마지막 값으로 즉시 조회
• 오래된 캐시 경고 배지 — 30분이 지난 값은 ⚠️로 명확히 표시
• 즐겨찾는 통화 쌍 최대 20건 저장
• 최근 조회 10건 자동 저장 및 탭 한 번으로 복원
• 통화 쌍 스왑 버튼 — 원화→달러를 달러→원화로 즉시 전환
• 빠른 통화 검색 — 코드 또는 이름으로 필터
• 광고·추적·분석 SDK 일체 없음
• 다크 모드 자동 대응

개인정보는 수집하지 않습니다. frankfurter.app에는 통화 코드만 전송되며,
입력한 금액·기기 정보·위치는 일절 서버로 전송되지 않습니다.
```

**자세한 설명** (English):

```
Currency Exchange converts 30+ currencies in real time and caches rates for
30 minutes so you can keep converting even when offline. A tiny, focused
finance utility — no ads, no trackers, no account.

Features
• 30+ currencies — official European Central Bank data via frankfurter.app
• 30-minute rate cache — keeps working when the network drops
• Stale-cache badge — rates older than 30 min are clearly marked with ⚠️
• Save up to 20 favorite currency pairs
• Auto-saves your last 10 lookups for one-tap restore
• Pair swap button — flip USD→KRW to KRW→USD instantly
• Fast currency search — filter by code or name
• No ads, no trackers, no analytics SDKs
• Automatic dark mode

No personal data is collected. Only the currency codes are sent to
frankfurter.app — the amount you enter, your device identifier, and your
location are never transmitted.
```
